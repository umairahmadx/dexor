# DevTools Hub

> Offline-first, 100% local-processing developer utility suite for iOS, Android, Web, macOS, Windows, and Linux.

## ✅ What's Working Now

### 15 Production-Ready Tools
- **JSON Formatter** — Format, minify, validate JSON
- **Base64 Encoder** — Text ↔ Base64 conversion
- **URL Parser** — Decompose and reconstruct URLs
- **HTML Entities** — Escape/unescape markup
- **Regex Tester** — Pattern matching with live feedback
- **JWT Decoder** — Decode tokens into header/payload/signature
- **YAML ↔ JSON** — Format conversion
- **String Hasher** — MD5, SHA-1, SHA-256, SHA-512 hashing
- **Password Generator** — Cryptographically secure password generation
- **Case Converter** — 7 case styles (camelCase, snake_case, etc.)
- **Word Counter** — Character, word, line, sentence, paragraph counting
- **Lorem Ipsum** — Configurable placeholder text generator
- **Hex Converter** — Text ↔ Hex encoding
- **Binary Converter** — Text ↔ Binary encoding

### Features
- 🎨 **Themeable** — Light/Dark mode, 5 accent colors, font scaling, compact mode
- 🔍 **Command Palette** — Search any tool by name or category
- ⚡ **Responsive** — Desktop sidebar, tablet drawer, mobile navigation
- 🔐 **100% Offline** — Zero network calls, all processing on-device
- 🧬 **Extensible** — 51 tool routes with placeholder scaffolds for remaining implementations

## Quick Start

```bash
flutter pub get
flutter run
```
